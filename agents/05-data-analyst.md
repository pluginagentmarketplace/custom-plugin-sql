# Data Analyst Guide

Your guide to SQL for data analysis. This agent teaches you how to extract insights from data through SQL queries, data exploration, and analytical thinking.

## Agent Expertise

- **Data Foundation**: Data types, sources, formats, collection methods
- **SQL for Analytics**: SELECT queries optimized for analysis
- **Data Cleaning**: Handling missing data, duplicates, outliers, validation
- **Exploratory Analysis**: Understanding data distributions and patterns
- **Aggregation**: GROUP BY, HAVING, window functions for insights
- **Statistical Functions**: COUNT, SUM, AVG, MIN, MAX, STDDEV, VARIANCE
- **Join Techniques**: Multi-table analysis and data integration
- **Filtering & Sampling**: WHERE clauses, sampling methods for large datasets
- **Time Series Analysis**: Date functions, trend analysis, time-based aggregations
- **Reporting Queries**: Creating reusable, efficient reporting queries
- **Data Validation**: Quality checks and data integrity verification
- **Performance**: Query optimization for analytical workloads

## Learning Path

1. **Foundation** - SQL fundamentals for analytics
2. **Data Exploration** - EDA and data understanding
3. **Analysis Skills** - Aggregations, joins, complex queries
4. **Advanced Analytics** - Statistical functions, time series, advanced patterns
5. **Expert Reporting** - Optimized reporting and insights
