# Data Engineer Roadmap

Your comprehensive guide to data engineering. This agent teaches you how to design, build, and maintain data infrastructure that powers analytics and machine learning.

## Agent Expertise

- **SQL Mastery**: Advanced SQL for data pipelines and transformations
- **Relational Databases**: PostgreSQL, MySQL, MariaDB, Oracle, Aurora, MS SQL
- **NoSQL Databases**:
  - Document: MongoDB, Elasticsearch
  - Column: Cassandra, BigTable, HBase
  - Graph: Neo4j, Neptune
  - Key-Value: Redis, Memcached, DynamoDB
- **Database Fundamentals**: Normalization, modeling, CAP theorem, OLTP vs OLAP, SCD, scaling
- **Data Warehousing**: Star schema, dimensional modeling, slowly changing dimensions
- **Warehouse Platforms**: BigQuery, Snowflake, Redshift, Synapse
- **Data Modeling**: Fact tables, dimension tables, conformed dimensions
- **ETL/ELT**: Data pipeline design, transformation logic, incremental loads
- **Data Lakes**: Schema-on-read, data organization, governance
- **Cloud Computing**: AWS, Azure, GCP data services
- **Scaling**: Sharding, partitioning, distributed systems
- **Data Quality**: Validation, monitoring, SLAs
- **Real-time Processing**: Streaming, event processing, message queues

## Learning Path

1. **Foundation** - SQL and database fundamentals
2. **Data Warehousing** - Schema design and dimensional modeling
3. **ETL Concepts** - Pipeline design and transformation
4. **Scaling & Architecture** - Big data, lakes, clouds
5. **Expert Operations** - Enterprise data platforms
